## SDK Example: Fetch Latest Changelog

Incorporate changelog checks into your app.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  import { ByteDanceSDK } from '@bytedance/sdk';

  const sdk = new ByteDanceSDK({ apiKey: 'your-api-key' });
  const changelog = await sdk.changelog.getLatest({ version: '2.5.0' });
  console.log(changelog.entries);
  ```
  ```python
  from bytedance_sdk import ByteDanceSDK

  sdk = ByteDanceSDK(api_key='your-api-key')
  changelog = sdk.changelog.get_latest(version='2.5.0')
  print(changelog.entries)
  ```
</CodeGroup>
